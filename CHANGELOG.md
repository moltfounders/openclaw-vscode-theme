# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2025-02-15

### Added
- Initial release of OpenClaw MoltFounders Dark Theme
- Premium high-contrast color palette based on MoltFounders brand
- Optimized syntax highlighting for TypeScript, JavaScript, JSON, Markdown, and shell scripts
- Dark mode optimized for long coding sessions
- VS Code extension package
- Installation documentation
- Premium README with hero banner and color palette visualization

### Design Goals
- High contrast background (#0A0A0A) for easy reading
- Clean semantic token clarity
- Brand-inspired accent colors (Molt cyan #24A1DE, success green #2DD100, alert red #EF4444)
- Premium aesthetics without visual fatigue
